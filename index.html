Welcome to Concentration

Three powerful tools — hover or tap each for details:

  • ▦ Eisenhower Matrix + Pomodoro
    Plan tasks by urgency & importance in the 4-quadrant matrix. Tap a task to start a 25-min Work → 5-min Break cycle (15-min long break every 4 rounds).
  • ▥ Vertical Time Gauge
    Calendar-style timeline of your day. See exactly when you worked on what; click any colored band to edit its start/end times.
  • ● Focus Score
    Measure your focus level by external factors (internet, devices, location) and internal factors (sleep, emotional wellness, energy). Pixel-art houses & faces visualize your score.

Concentration — User Manual

Preview

  • Top bar — global controls available from any page.
    • Motivation slot — a random quote from your list, re-picked every hour. Tap the quote to edit the list (one per line). When Focus Music is playing or you skip tracks, the slot briefly shows Focus Music: filename.mp3 for 3 seconds.
    • Sound & music controls — five circular pill buttons, all the same shape:
      • 🔈 / 🔉 / 🔇 — sound mode toggle (loud · soft · none).
      • ⏮ ▶ ⅠⅠ ⏭ — Focus Music playback. Tracks live in audio/focus/; the playlist is read from audio/focus/playlist.json.
    • Page tabs — Focus and Statistics. On phone, both shrink to matched pill icons (italic ƒ / Σ). The Setup page was consolidated into Focus; there is no Setup tab.
    • Manual — opens this guide. It also lives inside the Account panel (Sign out · Manual line).
  • Focus page — Money & Mana cards (top)
    • Money — your USD balance. Starts at $200 and only moves when something explicitly adds or removes money. W / D / H toggle compares the current balance to last week / yesterday / one hour ago. The "I'm doing" ticker and the Sleep button live inside this card.
    • Mana — willpower from −100 to +100. Visualized as a leaf (20 levels: withered → lush) and a horizontal score bar. The bar is draggable — drag the marker to set Mana directly; the leaf and mood text update live.
    • Time goals — under the Mana card, two clickable rows: Get things done by and Sleep at. Click either to open a popup: tap +1h / +30m / +10m to bump the time, Current Time to start from now, or Clear to wipe the goal. Set goals show as colored lines on the vertical gauge.
  • Focus page — Tasks (Eisenhower matrix)
    • Four quadrants colored by quadrant: Important · Urgent (pink), Important · Not Urgent (yellow), Not Important · Urgent (green), Not Important · Not Urgent (blue). Same colors paint the bands on the vertical gauge and the weekly view.
    • + add task — every cell has its own button. Tapping it creates a row and immediately opens the name editor.
    • ✎ pen next to a task — opens an editor where you can rename or delete the task (red Delete button).
    • $N pill — that task's bid (money you earn when you check it done, or lose when you uncheck it). Tap it to open a popup with +$10 / +$5 / +$1, −$1 / −$5, and ↺ Reset (back to the default in Edit Earning for Tasks). No free-form input — every change is in fixed steps.
    • Drag handle ≡ (PC only) — drag to reorder inside a quadrant.
    • Phone layout toggle — the small ⤢ / ⤡ button next to the "Tasks" title flips between the default 4-cell vertical stack and a swipeable 2×2 grid. Tapping the title text instead toggles the whole panel collapsed.
  • Pomodoro — tap any task name to start.
    • Cycle: 25m Work · 5m Good Break · 25m Work · 5m Good Break · 25m Work · 15m Longer Break, then repeats.
    • At each phase end a popup asks "Start <next phase>?":
      • Yes — advance to the next phase.
      • No after a Work phase — skip the break and start the next Work session immediately.
      • No after a Break — stop and log the whole pomodoro.
    • The popup syncs across devices: answer Yes/No on one device and the popup on every other device closes within ~1 second.
    • Tap the inline timer chip to pause / resume — paused time is excluded from the activity log and money awards.
    • While a pomodoro is running on a task, the bid pill is hidden on phone (timer fills the row); on PC, timer + bid sit side-by-side on a single line.
  • Money Moments & Mana Moments
    • Two columns of "moment" rows. Money Moments rows affect your Money balance; Mana Moments rows affect Mana. A ↔ button on each row moves it between sides (semantics flip with it).
    • Two row types per column:
      • Score chips — tap to apply a fixed change. Double-tap the name or score to edit.
      • Trackers (Internet browsing, Youtube/Netflix, Unrelated news, …) — tap to start a stopwatch, tap again to stop. Default rate is −3 / 5 min. Double-tap the rate to edit.
    • Sleep (0–10) and Emotional Wellness (−7…+7) sliders live inside the Mana Moments column. Both feed Mana at ×3.
    • Default trackers: Internet browsing and Youtube/Netflix in Money Moments; Unrelated news in Mana Moments.
  • Activity log (Focus page; collapsed by default)
    • Each row: type badge · task name · journal note (if any) · time range · money/mana changes · ✎ note button · ✕ delete.
    • Click the time range to edit start / end times.
    • Click ✎ (or the inline note text) to open the journal popup, which also includes datetime-local fields for editing the start / end time. Cancel discards everything; OK applies note + times atomically.
    • Money badges are Money; Mana badges use the muncher style.
    • On PC, the log splits into two columns (Money | Mana). On phone, it stays a single combined list.
  • Edit Earning for Tasks — a 2×2 grid of editable money rules.
    • Add task (+$5) — earned the first time you give a task a name.
    • Finish task (+$10) — default bid; per-task bids override.
    • Per minute (+10¢) — accrued during pomodoros and trackers.
    • Sleep skipped (−$30) — auto-applied if midnight passes without you clicking Sleep.
  • Account — a single horizontal row of 4 buttons:
    • Sign out
    • Manual — open this guide.
    • Device → Cloud — push this device's state to Supabase right now (use before clearing browser data).
    • Cloud → Device — pull the latest cloud state, overwriting this device.
  • Vertical time gauge (always visible on the left of the Focus page; never collapsible)
    • 24 hours of vertical scroll. Set / 2 / 4 toggle (under the gauge) controls the top edge: most-recent 4 AM, 2 hours ago, or 4 hours ago.
    • Red line + badge = current time. Blue line = "Get things done by" goal. Green line = "Sleep at" goal.
    • Colored bands as timers complete. Pomodoros use the task's quadrant color (pink / yellow / green / blue). Trackers use the sign of their rate: positive = pink, negative = light blue.
    • Click any band to edit its start / end time.
  • Statistics — Daily view
    • ← / → / Today nav at the top.
    • Tasks completed — laid out as the same Eisenhower matrix.
    • Money chart with W / D / H toggle — cumulative balance over the chosen window.
    • Mana score — line chart of every Mana snapshot for the day; click any dot for the exact time and value.
    • Activity log for the day — same row format as on the Focus page.
  • Statistics — Weekly view
    • Period header (← / → to move between weeks) shows the visible Mon–Sun range.
    • Activity Notes panel — shown between the period header and the gauge grid, listing every entry from the visible week that has a journal note. Each row carries a small uppercase date prefix (e.g. MAR 24 · 14:32 → 15:18 · 0:46:00). The whole panel hides itself when the week has no notes.
    • Below: 7 columns (Mon–Sun), each a 24-hour gauge with the same band colors as the Focus gauge. Sessions that cross midnight split correctly across two days.
  • Cloud sync & account
    • Email + password sign-up; session persists across reloads.
    • Try it → on the welcome screen lets you use the app without an account — data lives only on this device.
    • When signed in, every meaningful change pushes to Supabase: tasks, time goals, mana & money state, activity log, motivational quotes, sound mode, and so on. Other devices poll every 10 s (every 1 s while a confirm dialog is open).
    • After a fresh sign-in (cleared cookies, new browser), the app forces a cloud-wins pull so your tasks aren't overwritten by an empty default state.
    • Erase all data (Statistics page) wipes local + cloud after confirmation. Delete Account removes the Supabase account itself.
  • Collapsible panels — every titled section in the Focus page right column collapses on title click.
    • A small chevron (▾ / ▸) to the right of each title shows the state.
    • Activity log starts collapsed by default; everything else starts expanded. Your choice persists per device.
    • The vertical time gauge does not collapse.
  • Behavior notes
    • Mana never drifts on its own. It only moves when you toggle a factor, drag the score-bar, tap a Mana Moments row, or run a tracker / pomodoro that has a configured effect.
    • Money never drifts either, except for the once-per-day midnight check that applies the Sleep skipped penalty when the day ends without a Sleep tap.
    • The Tasks matrix and time goals reset at 3 AM each day (so you start the new day fresh) — money, mana, and the activity log are preserved.
